# GURPS Traveller - Notas

## Recomendaciones

Para una tripulación de una astronave, recomiendo la siguiente selección de habilidades:

> [!NOTE]
> Las habilidades que aparecen en el mismo recuadro son alternativas.

| Habilidad | Requisitos | Rol Tripulante |
|:----------|:-----------|:----|
| Spacer | -- | Cualquiera |
| Computer Operation | -- | Cualquiera, Especialista |
| Electrician | -- | Técnico, Ingeniero |
| First Aid | -- | Cualquiera, Médico |
| Mechanic (Maneuver Drive)<br>Mechanic (Jump Drive) | -- | Técnico, Ingeniero |
| Mechanic (High-Performance Spacecraft) | -- | Técnico |
| Mechanic (Fission Reactor)<br>Mechanic (Fusion Reactor) | -- | Técnico, Ingeniero |
| Navigation (Space) | -- | Navegante, Piloto |
| Navigation (Jumpspace) | -- | Navegante, Piloto, Capitán |
| Physician | -- | Médico |
| Pilot (High-Performance Spacecraft) | -- | Piloto |
| Electronics Operation (Communications)<br>Electronics Operation (Sensors) | -- | Especialista |
| Electronics Repair (Communications)<br>Electronics Repair (Sensors)<br>Electronics Repair (Computer) | -- | Técnico, Especialista |
